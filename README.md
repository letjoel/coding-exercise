# Node Coding Excercise

## Requirements

Node.js v.18

## Dependencies

Using new native testing module from node 18.

## Install

```
npm install
```

## Scripts

### Testing

```
npm test
```

### Execute the app

```
npm start
```

A new depurated json will be created in 'output' folder. Discarded objects will be shown in console.
